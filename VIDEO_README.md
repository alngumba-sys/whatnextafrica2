## 🚀 Quick Start (3 Commands)

```bash
# 1. Make scripts executable
bash make-executable.sh

# 2. Deploy your platform (choose one)
./deploy-vercel.sh        # Permanent hosting
# OR
./deploy-ngrok.sh         # Quick temporary demo

# 3. Generate QR code (use your URL from step 2)
node generate-qr.js https://your-deployed-url.com
```

## 📱 QR Code Guidelines

- **Show for:** 5+ seconds (intro and outro)
- **Size:** 300x300px minimum on screen
- **Placement:** Top-right corner or full screen
- **Test:** Scan from your monitor before recording

## 💡 Pro Tips

1. **Practice first** - Record a test run
2. **Speak slowly** - 20% slower than normal
3. **Show features** - Don't just talk about them
4. **QR code** - Make it large and visible
5. **Edit ruthlessly** - Remove all mistakes
6. **Test QR** - Scan from your actual screen
7. **Background music** - Optional, keep very low

## 🐛 Troubleshooting

### Can't deploy?
→ Check `CONVEX_TROUBLESHOOTING.md`

### QR code not working?
→ Ensure URL starts with `https://`  
→ Test by scanning with phone

### Audio issues?
→ Check microphone permissions  
→ Test in recording software first

### Video too large?
→ Use H.264 codec  
→ Reduce bitrate to 8 Mbps
